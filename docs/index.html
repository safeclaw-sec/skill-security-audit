What It Detects

Six categories of threat analysis covering the most common attack vectors in AI agent skills.

🚨

Prompt Injection

Hidden instructions in SKILL.md to exfiltrate data, bypass security rules, or impersonate system prompts.

🔐

Data Exfiltration

Code that reads API keys, tokens, .env files, or credentials and sends them to external servers.

💣

Malicious Code

eval(), exec(), unsanitized subprocess, crypto miners, websocket listeners, post-install hooks.

🔍

Typosquatting

Dependencies with names similar to popular packages (e.g., "requets" instead of "requests").

🌐

Network Abuse

Connections to suspicious domains like ngrok, requestbin, or undocumented external APIs.

🕵

System Reconnaissance

Collection of hostname, IP address, username, platform info, or filesystem structure.

Quick Start

No API keys. No installation. Just Python 3.8+.

# Clone the repo
git clone https://github.com/safeclaw-sec/skill-security-audit.git

# Audit a local skill
python3 scripts/audit.py /path/to/any/skill

# Audit with offline mode (no ClawHub lookup)
python3 scripts/audit.py /path/to/skill --no-clawhub

# Self-audit (verify integrity)
python3 scripts/audit.py . --no-clawhub
